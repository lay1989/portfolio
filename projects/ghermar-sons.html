Ghermar & Sons

Ghermar & Sons approached me to create a modern, professional website that would effectively showcase their import-export business. The goal was to create a landing page that immediately conveyed trust, professionalism, and global reach while maintaining fast loading speeds and mobile responsiveness.

Role

Lead Designer & Developer

Timeline

4 Weeks

Tools

HTML, JavaScript, Tailwind CSS, transitions & animations

Problem

The client needed a website that would showcase their portfolio without distracting from the work itself. The challenge was to create a design that fits everything on a single page while maintaining visual hierarchy and user engagement. The existing brand had a specific color palette that needed to be incorporated, and the content needed to appeal to both B2B clients and potential partners.

Approach & Solution

We adopted a 'radical reduction' approach, stripping away all non-essential elements. We used the brand's logo color palette with a single accent color to guide the user's eye. The typography is bold and expressive, acting as a primary design element. I implemented smooth scroll-triggered animations using vanilla JavaScript and CSS transitions to create an engaging user experience without compromising performance.
